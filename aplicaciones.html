Aplicaciones Web


Frontend

El frontend es aquello que ven los usuarios e incluye elementos visuales, como botones, casillas de verificación, gráficos y mensajes de texto. Permite a los usuarios interactuar con la aplicación. Las Tecnologías más utilizadas son:

  • HTML
  • CSS
  • JavaScript

Backend

El back-end son los datos y la infraestructura que permiten que la aplicación funcione.


Tecnologías Backend

Las tecnologías más utilizadas son:

  • Base de Datos
  • Servidores Web
  • Lenguajes (PHP, Python, Node.js )

FullStack

Un desarrollador FullStack es capaz de trabajar tanto en el frontend como en el backend.
